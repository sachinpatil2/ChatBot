    Lima bot is a next generation conversational interface and this section provides you a quick help on how to use Lima Bot.

    Login

    To access Lima Bot features, user will be asked to enter the login credentials. Please reach out to your administrator to get the user credentials.

    Lima UI Interface overview

    Here is a brief overview of Lima Bot interface.

    • 1 - The current user who is using the system.
    • 2 - Chat window that records the dialog between user and bot.
    • 3 - Task information window shows the ongoing tasks and the entities within the tasks that are being captured through conversation.
    • 4 – Self-service window that shows the additional information that bot expects user to refer to. These could be document, image, videos, maps etc.

    Interaction Modes

    Lima bot supports two modes of interaction

    • a. Text Mode
    • b. Voice Mode

    Text Mode

    This is mode where user uses the chat window to communicate with Bot.
    The Manual Mode does not apply any spell check feature and sends text as is entered by user.
    The Auto Mode uses auto spell check and word segmentation feature before it is sent to Bot.

    Voice Mode

    This is mode where user uses the speech recognition feature to communicate with Bot. The Manual Mode requires user to click on “MIC” button whenever he wants to record his utterance. In Auto Mode “MIC” button is activated automatically whenever Bot wants user to provide his/her input. In case user is not able to record his/her utterance he/she can manually click the “MIC” button and continue the conversation. The system speakers are enabled only in “Voice” mode so that user and bot interaction is completely through audio signals.

    Other Features

    Clear

    User can clear the earlier conversation by pressing clear button

    Logout

    User can logout of the conversation by pressing Logout button
